# Divisor de Sílabas
Uma aplicação web feita com Python, Django, Bootstrap, dentre outras bibliotecas que recebe uma palavra qualquer da Língua Portuguesa e retorna sua divisão silábica (fazendo web scraping do site <a href="https://www.dicio.com.br/" target="_blank" rel="noopener noreferrer">**dicio.com.br**</a>), juntamente com algumas informações adicionais sobre a palavra. O website também possui um sistema de envio de e-mails para quaisquer dúvidas, feedbacks ou erros no funcionamento do programa.

- O site pode ser acessado em <a href="https://divisor-de-silabas.onrender.com" target="_blank" rel="noopener noreferrer">**divisor-de-silabas.onrender.com ↗**</a>
- Caso queira executar o site localmente, primeiro baixe os arquivos e abra um prompt de comando na pasta do projeto. Depois, com o Python instalado, execute `pip install -r requirements.txt` para instalar as dependências necessárias. Em seguida, execute o comando `python manage.py collectstatic` e logo após execute `python manage.py runserver`. Pronto, agora é só aproveitar!

> Nota do Dev: na versão original do projeto, a biblioteca PySimpleGUI foi usada para a criação de uma interface gráfica que atendesse aos requisitos da aplicação.
